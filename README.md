# Nikita Bhansali - Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Java Full Stack Developer.

## 🚀 Features

- **Modern UI/UX**: Sleek dark theme with gradient accents and smooth animations
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Powered by Framer Motion for engaging user experience
- **Interactive Sections**: 
  - Hero section with animated background
  - About section with education timeline
  - Skills showcase with tech stack icons
  - Projects gallery with GitHub links
  - Contact form with social media links

## 🛠️ Technologies Used

- **React** - Frontend framework
- **Vite** - Build tool and dev server
- **Framer Motion** - Animation library
- **React Icons** - Icon library
- **CSS3** - Styling with custom animations
